# Taxi Poreč Website - Project Overview

## Project Description
Premium taxi service website for Poreč, Croatia, designed to attract tourists and locals with professional design, excellent SEO, and easy booking options.

## Target Audience
- **Primary**: Tourists visiting Poreč and Istria region
- **Secondary**: Local residents needing reliable taxi services
- **Languages**: Croatian (primary), English, German, Italian

## Core Objectives
1. Generate direct bookings via phone, WhatsApp, and email
2. Rank highly in Google searches for "taxi Poreč", "taxi Porec", "airport transfer Poreč"
3. Build trust through professional design and clear information
4. Mobile-first experience (80% of tourists search on mobile)

## Key Features

### 1. Contact Methods
- **Phone Call Button**: Direct click-to-call functionality
- **WhatsApp Integration**: Instant messaging with pre-filled message
- **Email Contact Form**: Professional inquiry system
- **Visible Contact Info**: Always accessible in header/footer

### 2. Essential Information
- Service areas (Poreč, Istria, airport transfers)
- Pricing transparency
- Fleet information
- Driver credentials
- 24/7 availability
- Payment methods

### 3. SEO Requirements
- Semantic HTML5 structure
- Schema.org markup (LocalBusiness, TaxiService)
- Optimized meta tags
- Fast loading speed (<3s)
- Mobile responsive
- Local SEO optimization
- Croatian and English content

### 4. Design Principles
- **Premium aesthetic**: Clean, professional, trustworthy
- **No gradients**: Solid colors only
- **High contrast**: Excellent readability
- **Professional typography**: Modern sans-serif fonts
- **Quality imagery**: Professional photos of Poreč, vehicles
- **White space**: Generous spacing for premium feel

### 5. Responsive Design Requirements
- **CRITICAL**: Website MUST be fully responsive for both mobile and desktop
- **Mobile-first approach**: Design and develop for mobile screens first, then scale up
- **Breakpoints**: Support all screen sizes from 320px (mobile) to 1920px+ (desktop)
- **Touch-friendly**: All interactive elements must be easily tappable on mobile (min 44x44px)
- **Adaptive layouts**: Content must reflow gracefully across all device sizes
- **Performance**: Fast loading on mobile networks (3G/4G)
- **Testing**: Must test on actual mobile devices and desktop browsers

## Technical Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern styling, CSS Grid, Flexbox
- **Vanilla JavaScript**: Minimal, performance-focused
- **No frameworks**: Pure HTML/CSS for speed and SEO
- **Progressive enhancement**: Works without JavaScript

## Success Metrics
- Google PageSpeed score >90
- Mobile-friendly test pass
- First Contentful Paint <1.5s
- Ranking in top 3 for "taxi Poreč"
- High click-through rate on contact buttons

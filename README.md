# Taxi Poreč Website

Premium taxi service website for Poreč, Croatia. Built with HTML, CSS, and vanilla JavaScript for optimal performance and SEO.

## Quick Start

### Download Assets from Old Website

**Simple version** (recommended):
```bash
./download-assets.sh
```

**Detailed version** (shows more info):
```bash
./download-assets-v2.sh
```

This will download the logo, favicon, and hero images from the existing porectaxi.com website. The scripts handle errors gracefully - some images may not be accessible, which is normal.

See **[DOWNLOAD_INSTRUCTIONS.md](docs/DOWNLOAD_INSTRUCTIONS.md)** for detailed information.

## Project Overview

A professional, SEO-optimized website for a taxi service in Poreč, Croatia. The site features:
- Premium design with no gradients
- Excellent SEO optimization
- Multiple contact methods (phone, WhatsApp, email)
- **FULLY RESPONSIVE: Works perfectly on mobile phones, tablets, and desktop computers**
- Mobile-first responsive design approach
- Fast loading times
- Multilingual support (Croatian, English)
